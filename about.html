Welcome to TwitterOnTap!

What is TwitterOnTap?

This website was created as part of a software engineering project for the UCD MSc (Conversion) in Computer Science (2013/2014). The site incorporates a web application whose primary function is to grab recent tweets from Twitter and analyse their sentiment.

How does it work?

The Sentiment analysis tool is essentialy a java web-application which primarily incorporates and utilises the following Applcation Programming Interfaces (APIs):

  • Twitter
  • Twitter4j
  • Lingpipe
  • JFreeChart
  • mySQL Connector

The application takes in a user keyword as a query and then uses the Twitter4j API which communicates with the Twitter API to rip recent tweets from Twitter according to the search word. It then performs sentiment analysis on the tweets using a Machine Learning classifier which was trained with the use of the Lingpipe API. The results are then displyed with the help of the JFreeChart API. All of this functionality was then integrated into a web-interface using Java Server Pages (for the web-page integration) and Java Database Connection with the mySQL Connector API (for the database integration).
