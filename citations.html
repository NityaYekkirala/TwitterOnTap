CITATIONS

Twitter

The Twitter API allows applications and other APIs to interact with Twitter and send/collect data to/from it.

Twitter4j

Twitter4j is an unofficial java library for the Twitter API. With this library you can easily integrate your java application with the twitter service.

Lingpipe

Lingpipe is a toolkit for processing text using computational linguistics. It contains a java-based API.

Sananalytics

The Sandars Twitter Sentiment Corpus is a free data set for training and testing sentiment analysis algorithms. It consists of 5513 hand-classified tweets.

JFreeChart

JFreeChart is a free 100% Java chart library that makes it easy for developers to display professional quality charts in their applications.

MySQL Connector/J

MySQL offers standard database driver connectivity for using MySQL with applications and tools that are compatible with industry standards ODBC and JDBC. MySQL Connector/J is the official JDBC driver for MySQL.

Open CSV

opencsv is a very simple csv (comma-separated values) parser library for Java.
